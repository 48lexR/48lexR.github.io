Javascript Cheat Sheet

Javascript helps us implement reactivity and interactivity to our webpage.

Primitive Data Types & Variables

There are several "primitive" data types that are how data is encoded. Use typeOf() to return the type of a value.

Number: A number is a numerical value and is processed as such. A number can be mathematically changed.
String & Character: A character is a non-number encoded by surrounding it in quotes. A string is a list of characters.
Boolean: A boolean value is a value which is either true or false.

Some Common Operators.

+ is the Addition operator.
- is the Subtraction operator.
* is the Multiplication operator.
/ is the Division operator.
% is the Modulus operator.
= is the Assignment operator.

A variable is a keyword for a value, assigned by the keyword let or const. Variables can be manipulated by operators. For example, the following code adds two variables: 

                let num1 = 2;
                let num2 = 2;
                console.log(num1 + num2);
                (Expected output: 4)

Please note: The assignment operator (=) is not an "equals sign."

You can change between data types by using certain functions. For example, Number(variable/string) takes a variable or string and returns a number. This is particularly useful for user input that requires math, so you would need to store the data as a number, not a string. For example: 

                
                    let miles = prompt("Mileage:");
                    miles = Number(miles);
                    let km = miles * 1.609;
                    console.log(km);
will convert a prompted number into kilometers by converting miles into a number first.

Protip: Variable naming conventions tend to be

  1. Intuitive
  2. Easy to type
  3. Typed in Camel Case
  4. Started without numbers

Strings

Strings are also particularly useful. They can start with either " or ' or `, but it must close with the same one where it opens. The ` string can contain variables. For example: `Hello ${name}` accesses a variable "name."

Strings are stored as lists of characters, and they can be accessed like they are accessed in arrays. For example, in the string let getName = "Cage"; console.log(getName[1]); accesses the letter at index 1 of "Cage". That letter is a. Indexes always start at 0, never 1.

Strings have a ton of useful features, called methods and attributes. Methods are functions. For example, <str>.substring(a, b) returns all the characters between indexes a and b. <str>.length returns the number of characters in the string.

Functions

A function is a callable piece of data that gives the computer more instructions on what to do. For example: 



            
                function run(){

                     let name = prompt("What is your name?");

                     console.log("Hello " + name + "!");

                }
            
            
            

            
            
The code defines "run()" as a function that does two actions: prompt the user to enter a name
                (stored as name) and return that name in the console, along with a message: Hello [name]! A function
                always has a name, and takes any number of variables (stored inside the parentheses). All function calls
                include parentheses. 


            
A function definition looks like this:
            
                        function <name> (<args>, ...){
                            //logic goes here 
                            return value; // optional
                        }
                    

            

            
And a function call looks like this:
            
                        <name>(<args>, ...);

Logic relies on booleans and the evaluation thereof to create conditionals. These are
                very powerful in coding, because they help our programs become versatile.

            
For example, || returns true if one of the conditions on either side of it is true, but
                && returns true only if the conditions on both sides are true. ! will invert whatever follows it. 

            
Conditionals are particularly useful in if-statements to direct the flow of our program. An if-statement
                only evaluates the code within it if a condition is met.

            
An if-statement can also contain an "else" statement, that will only run if the rest of the if statement
                is untrue. An if-else statement combines both of these, so the code will go through and check all the
                conditions before finding the code it needs to run. It will only test one condition at once.


            
The Document Object Module: Making Webpages Interactive

            
Javascript is primarily used to make webpages interactive, and this is done through the "document
                object module" or DOM. The DOM is an object‐a data type which has both properties and methods
                linked to the main object by periods‐whose properties are listed in the order which
                they appear. For example, to access the <body> tag of the document, the javascript
                must be: document.body, which will return the "body" as an object.

            
It is customary to access elements based on their class (using
                document.getElementsByClassName(*args)) or id (using
                document.getElementById(args). The former will return a list of elements
                that all have the class name specified in args, and the latter will return a single element that has the
                id specified in args.
